The G3_circuit matrix exceeds GitHub's 100MB file size limit. To run benchmarks with this dataset, please download it manually from the SuiteSparse Matrix Collection:

- **Format**: Matrix Market (MM)
- **Download Link**: [SuiteSparse G3_circuit](https://www.cise.ufl.edu/research/sparse/matrices/AMD/G3_circuit.html)

Once downloaded, extract the `.mtx` file and place it in deliverable2 folder.
